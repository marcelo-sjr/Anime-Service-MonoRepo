# Anime-Service-MonoRepo
O objetivo do App é construir uma lista de de animes, onde cada usuário poderá configurar animes que são de seu interesse, assistidos, etc.

Não disponibilizei o jar da biblioteca commons-core, portanto para startar o serviço, é necessário fazer um 'mvn clean install' na root do Commons-Core, para que este seja buildado e colocado em sua pasta .m2
